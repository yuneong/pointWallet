## 📡 API 리스트

### 1. `사용자 포인트 지갑 생성`
- POST
- /api/wallet

### 2. `사용자 포인트 지갑 조회`
- GET
- /api/wallet/{walletId}

### 3. `사용자 포인트 충전`
- POST
- /api/wallet/{walletId}/charge

### 4. `사용자 포인트 사용`
- POST
- /api/wallet/{walletId}/usage

### 5. `포인트 내역 조회`
- GET
- /api/wallet/{walletId}/history

### 6. `지갑 잔액 조회`
- GET
- /api/wallet/{walletId}/balance
